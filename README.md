# claude-code-playground

A sandbox for running Claude Code (Sonnet 4.6) through its paces alongside [RTK](https://github.com/example/rtk) (Rust Token Killer) for token-optimized development workflows.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Running tests

```bash
pytest
```
